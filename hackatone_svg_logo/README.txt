Hackatone SVG Logo Assets

Files included:
1. hackatone_icon.svg
   - Icon mark only.
   - Use for favicon, small UI mark, social profile mark, loading states.

2. hackatone_wordmark.svg
   - Text wordmark only.
   - Uses a rounded sans-serif font stack. Text remains editable.

3. hackatone_logo_horizontal.svg
   - Full horizontal logo lockup.
   - Use for website header, landing page, presentation cover, documents.

4. hackatone_logo_stacked.svg
   - Stacked square-friendly logo layout.
   - Use for posters, app splash screen, square social posts.

5. hackatone_app_icon.svg
   - Rounded-square app icon version.
   - Use for mobile app icon drafts, web app icon, PWA icon concept.

Design notes:
- Colors follow the Hackatone design system: #FF8A3D, #E96F26, #FFD166, #FFF8EF, #2B2B2B.
- SVG groups are divided with IDs such as icon-mark, wordmark, app-tile, and digital-spark.
- Wordmark is editable text, not outlined. If you need exact visual consistency across all devices, convert the text to outlines in Figma/Illustrator after choosing the final font.
